### Step 5: Create GitHub Repository

1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right
3. Select "New repository"
4. Repository name: `shl-assessment-recommendation-system` (or your preferred name)
5. Description: "Intelligent SHL Assessment Recommendation System using semantic search with Gemini embeddings"
6. Visibility: **Public** (for assignment submission)
7. **DO NOT** initialize with README, .gitignore, or license (we already have these)
8. Click "Create repository"

### Step 6: Connect Local Repository to GitHub

```bash
# Add remote repository (replace with your GitHub URL)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Verify remote
git remote -v
```

### Step 7: Push to GitHub

```bash
# Push to main branch
git branch -M main
git push -u origin main
```

### Step 8: Verify Upload

1. Go to your GitHub repository
2. Verify all files are present
3. Check that sensitive files (`.env`, `venv/`, etc.) are NOT visible
4. Verify README.md displays correctly
